# Profile showcase

A minimal, privacy-first portfolio showcase for GitHub profiles. It renders a concise, professional README-style summary from a YAML configuration without any personal identifiers.

## Features
- Modular sections: headline, skills, projects, highlights, and contact-neutral badges
- Privacy-first: no names, emails, or personal data; purely technical content
- Quick customization via `config.yml`
- Deterministic Markdown output for clean commits
